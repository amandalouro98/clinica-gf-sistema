# Clínica Gestão – Pacote rápido (Windows)

## Como usar
1. Extraia **este ZIP** dentro da pasta do seu projeto (ex.: `C:\Users\SEU_NOME\Desktop\sistema GF`).
   - Se pedir para substituir arquivos, pode clicar em **Substituir**.
2. Abra o **Terminal** dentro da pasta do projeto (na barra de endereço digite `cmd` e Enter).
3. Ative o ambiente virtual:
   ```bat
   .venv\Scripts\activate
   ```
4. Rode a aplicação:
   ```bat
   streamlit run app.py
   ```
5. Login inicial:
   - Email: `admin@clinica.com`
   - Senha: `Admin@123`
